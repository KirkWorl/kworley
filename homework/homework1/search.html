Machine Learning and Its Role in Search

Machine learning plays a core part at almost every part of the stack at large search engines like Google or Bing.
Machine learning plays an important role in search engines, and is currently one of the biggest avenues in which machine learning is making progress. It is utilized in search rankings, where systems can learn how to better rank queries and results via different types of search rankings. Machine learning is also embedded in understanding any particular query; It helps address problems like query classification, spelling suggestions, query expansion, and intent disambiguation. Another area it assists in is search features. These can be things like site links, knowledge graphs, and items other than the actual results. Yet another area machine learning is crucial in, and one of the biggest functions of a search engine is web crawling. To index webpages and make them accessible search engines first have to 'crawl' the web, that is, visit webpages and take snapshots. Machine learning helps systems learn how often to crawl a particular site, and the optimal manner in which to do so. These are just a couple of areas in which machine learning has made its mark on search engines, and will continue to develop in the future.
